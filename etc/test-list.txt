#

#   - number

#   - count of elements into a database
#   - starting date
#   - ending date
#   - commit's sha1 (see script info.sh)
#   - some info about hardware and software (see script info.sh).
#   
#   Comments:
#	Result of this benchmark has some specific. 
#	Benchmark is run 50 times and then ru.petrsu.nest.yz.benchmark.benchmark/avg (from utils.clj) count average value.
#	The bench_for_file.sh is:
#	  for n in "1000 5000 10000 15000 20000 50000 100000"; do
#	    java -cp $CP clojure.main --main ru.petrsu.nest.yz.benchmark.bench-norepl "nest.mom" $n 1 "test-list.txt" "list"
#	  done;
#	And then:
#	  for i in `seq 1 50`; ./bench_for_file.sh; done;
#
#	Also parsing time is not counted.
